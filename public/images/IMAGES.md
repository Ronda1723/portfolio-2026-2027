# Image drop-in checklist

Every image on the site is a **placeholder** until you add the matching file here.
Drop a PNG at the exact path below and it appears automatically — no code changes.
(Dimensions are a recommendation; any matching aspect ratio works.)
Click any placed image on the site to open it in the Picture Viewer.

Case studies follow the long-form structure:
Hero → Executive Summary (before/after) → Problem → Research → Opportunity →
Solution (3 decisions) → Collaboration → Launch → Results.
